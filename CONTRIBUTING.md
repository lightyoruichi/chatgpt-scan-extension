# Contributing

We welcome contributions to the ChatGPT Search Query Revealer project!

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone your fork** to your local machine:
    ```sh
    git clone https://github.com/YOUR_USERNAME/chatgpt-search-revealer.git
    cd chatgpt-search-revealer
    ```
3.  **Create a new branch** for your feature or bug fix:
    ```sh
    git checkout -b my-feature-branch
    ```
4.  **Make your changes** and commit them with valid commit messages.
5.  **Push your changes** to your fork:
    ```sh
    git push origin my-feature-branch
    ```
6.  **Open a Pull Request** against the main repository.

## Development

To test your changes locally:
1.  Go to `chrome://extensions/`.
2.  Enable **Developer mode**.
3.  Click **Load unpacked** and select the extension directory.
4.  Visit [chatgpt.com](https://chatgpt.com) to verify functionality.
